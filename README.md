# Sistema de Feedback com MongoDB

Projeto acadêmico desenvolvido com foco em banco de dados NoSQL, modelagem orientada a documentos e análise de feedbacks de aplicativos móveis.

## Objetivo

Desenvolver uma solução NoSQL para armazenar, consultar e analisar feedbacks de usuários de aplicativos móveis, permitindo flexibilidade no armazenamento de dados semiestruturados.

## Tecnologias utilizadas

- MongoDB
- NoSQL
- JSON
- Modelagem de Documentos
- Git
- GitHub

## Estrutura do projeto

```txt
docs/
images/
scripts/
```

## Funcionalidades implementadas

✅ Modelagem orientada a documentos  
✅ Estruturação de dados em JSON  
✅ Consultas por filtros  
✅ Aggregation pipeline  
✅ Análise de avaliações de usuários  

## Análises realizadas

- Aplicativos com melhores avaliações
- Usuários mais ativos
- Comentários por categoria
- Distribuição de notas
- Tendências de feedbacks

## Comparativo arquitetural

Durante o projeto foram analisadas diferenças entre:

- Bancos relacionais
- Bancos NoSQL
- Dados estruturados vs semiestruturados
- Escalabilidade horizontal

## Aprendizados

Neste projeto foram aplicados conceitos de modelagem NoSQL, manipulação de documentos JSON, aggregation pipeline e análise de dados em ambientes flexíveis.
